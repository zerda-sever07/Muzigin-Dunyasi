# 🎵 Müzik Dünyası

Modern ve kullanıcı dostu bir arayüz ile geliştirilen bu web uygulaması, kullanıcıların müzikleri kolayca keşfetmesini, filtrelemesini ve etkileşimde bulunmasını sağlar.

---

## 🚀 Özellikler

- 🔍 **Gelişmiş Arama Sistemi**
  - Şarkı adına göre arama
  - Sanatçıya göre arama

- 🎯 **Filtreleme Seçenekleri**
  - Müzik türüne göre filtreleme (Pop, Rock, Rap vb.)
  - Dil seçeneğine göre filtreleme
  - Sanatçı bazlı listeleme

- 💬 **Yorum Sistemi**
  - Kullanıcılar şarkılar hakkında yorum yapabilir
  - Mevcut yorumları görüntüleyebilir

- 🎨 **Şık ve Modern Tasarım**
  - Responsive (mobil uyumlu) arayüz
  - Kullanıcı deneyimi odaklı tasarım

- HTML
